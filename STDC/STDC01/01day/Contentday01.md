  一、绝对路径的写法：
  1.直写法：(:/abcvbn[它表示了:从根目录到vbn的一条路线])
  2.分隔法：(:/abc/vbn[它表示了:从根目录到vbn的一条路线])

  二、绝对路径的例子：  
  1.root -> {abc def} :/abcdef(:/abc/def)
  2.root -> abc :/abc
  3.root -> def :/def
  4.root -> abc -> asd  :/abc/asd(:/abcasd)
  5.root -> abc -> iop  :/abc/iop(:/abciop)
  6.root -> abc -> vbn  :/abc/vbn(:/abcvbn)
  7.root -> abc -> {asd iop vbn}
  :/abc/asd(:/abcasd)
  :/abc/iop(:/abciop)
  :/abc/vbn(:/abcvbn)
  8.root -> def -> pqr :/def/pqr(:/defpqr)
  9.root -> def -> xyz :/def/xyz(:/defxyz)
  10.root -> def -> {pqr xyz}
  :/def/pqr(:/defpqr)
